## parser

parse output of sysbench for MySQL.  
These items are extract from sysbench output.

- version of sysbench
- version of LuaJIT
- Number of threads
- Number of total read/write/other/Tx
- TPS/QPS
- Ignored errors
- Reconnects
- Total Time/Events
- MIN/AVG/MAX/95th/SUM Latency
- ThreadsEvent AVG/STDDEV
- ThreadsExecTime AVG/STDDEV
